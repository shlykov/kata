# Bootstrap repository for tdd katas
[About katas] (http://codekata.com/)
## Using

**Init** 

` php -r "readfile('https://getcomposer.org/installer');" | php `

` php composer.phar install `



**Run tests**

Auto

` gem install watchr rev ruby-fsevent`

` watchr whatch.rb `

Or manual

` phpunit `
